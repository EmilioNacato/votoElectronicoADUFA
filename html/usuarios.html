GESTIÓN DE USUARIOS

GESTIÓN DE USUARIOS

En este apartado, podrá gestionar de manera integral los usuarios del sistema, incluyendo la creación, edición, y eliminación de cuentas. Podrá asignar roles y permisos específicos y asegurarse de que la información de cada usuario esté actualizada y correcta. Esta funcionalidad es esencial para mantener la seguridad y organización del sistema, permitiendo un control efectivo sobre el acceso y las acciones de cada usuario dentro de la plataforma.



Carga Masiva de Usuarios

Instrucciones:

  • El archivo debe estar en formato CSV o Excel (.xlsx)
  • Las columnas requeridas son:
    • NOMBRE_US (Nombres completos, ejemplo: "Emilio Julián")
    • APELLIDO_US (Apellidos completos, ejemplo: "Ñacato Ortega")
    • DEPARTAMENTO_US (Departamento al que pertenece)
    • CONTRASENA_US (Contraseña para el usuario)
    • ID_ROL (Opcional, por defecto será 2 - votante)
  • El archivo debe estar codificado en UTF-8 para manejar correctamente tildes y ñ
  • El sistema generará automáticamente:
    • ID_US: usando las iniciales del nombre y el primer apellido (ejemplo: "ejnacato" para "Emilio Julián Ñacato Ortega")
    • Correo electrónico: ejnacato@espe.edu.ec
    • Estado: activo (1)
  • Puede descargar archivos de ejemplo:
    • Descargar ejemplo CSV
    • Descargar ejemplo Excel
